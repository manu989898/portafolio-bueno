Caja Registradora

Descripción del Proyecto

La Caja Registradora es una aplicación diseñada para facilitar la gestión de ventas en pequeños negocios. Ofrece una interfaz intuitiva y funciones esenciales para el manejo eficiente de transacciones diarias.

Características principales:

  • Gestión de reservas
  • Impresión de ticket
  • Generación de informes de ventas
  • Interfaz amigable y fácil de usar
